Display your Wordpress galleries as full width slideshow. Requires the "Simplest Gallery" plugin (adds a new gallery style to it).


== Description ==


Adds a new gallery style to the Simplest Gallery plugin, producing a nice full width slideshow.

The slideshow should adapt to responsive pages automatically, and it shows up quickly: as soon as the first image is loaded.

It requires the "Simplest Gallery" plugin to work.





== Installation ==


1. Install from the WP administration interface: Plugins->Add New, and Activate
2. Go to Settings/Simplest Gallery, select jQuery Cycle Slideshow from the list, save changes.
3. View any page with a gallery and enjoy

For best results, resize images to the same size before using them in a gallery.
For more tips and help/support, check out the plugin's web page: [Plugin page](https://www.simplestgallery.com/add-ons/jquery-cycle-slideshow-gallery-style-plugin/ "jQuery Cycle Slideshow plugin page")

4. OTIONAL - Use php code to include a slideshow (slider-style) in any page - learn how to do this on this demo site: https://www.simplestgallery.com/homeslider/
